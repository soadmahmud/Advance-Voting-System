# Advance Voting System

A desktop-based digital voting system built with C#, Windows Forms (.NET) and a relational DBMS. Developed as an academic project at Bangladesh University of Business and Technology (BUBT) to demonstrate secure voter registration, candidate management, and administrative controls.

## Features

* **Voter Panel**: Allows voters to authenticate and cast their vote securely.
* **Admin Panel**: Manage the election cycle, including adding new candidates, monitoring voting, and viewing real-time results.
* **Registration System**: Secure registration process for new voters with data validation.

## Technologies Used

* **Language:** C#
* **Framework:** Windows Forms (.NET Framework 4.8)
* **Database:** SQL Server (SQL Express)

## Setup & Installation

To run this application locally, you will need Visual Studio and an instance of SQL Server.

1. Clone the repository:
   `git clone https://github.com/soadmahmud/Advance-Voting-System.git`
2. Open the `Voting System (SDP).sln` file in Visual Studio.
3. **Database Setup:** Ensure your local SQL Server is running. If needed, create the `SDP_Voting_System` database and update the connection string in `App.config` to point to your local server instance (currently points to `DESKTOP-NMF9NAR\SQLEXPRESS`).
4. Build and Run the project.
